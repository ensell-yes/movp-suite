# C7.1 Editor Dependency Spike — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Decide BlockNote vs TipTap for Phase C7 by proving, head-to-head, that a candidate can round-trip MOVP `content_revision.data` hash-stably through the real create → **editor-driven edit** → publish → published-read lifecycle, within the client/server boundary, at an acceptable license.

**Architecture:** A throwaway nested pnpm workspace `spikes/editor/` (outside the root workspace globs) with four packages: `@spike/fixture` (browser-safe data + the normative canonical algorithm), `@spike/oracle` (Node-only; runs the real `@movp/domain` hash path through a real `SupabaseClient` wired to a stateful fake `fetch`), and `@spike/blocknote` / `@spike/tiptap` (equivalent React editor islands + Playwright-mounts/Node-hashes proofs, boundary gates, a11y, and a machine-validated `CandidateResult`).

**Tech Stack:** pnpm 9.12.0 nested workspace; TypeScript (bundler resolution); Vitest (Node); Playwright + `@axe-core/playwright`; Vite (islands + `getModuleIds` boundary artifact); BlockNote (`@blocknote/*` + Mantine) vs TipTap (`@tiptap/*` + StarterKit).

**Source spec:** `docs/superpowers/specs/2026-07-15-c7.1-editor-spike-design.md`
(land this revised spec and plan together). Section refs below (§N) point at it.

## Global Constraints

- **Isolation, checked BEFORE every commit and against a base SHA.** `spikes/` is NOT in root `pnpm-workspace.yaml`. Task 1 records the pre-spike commit in `spikes/editor/.spike-base-sha`. Before every `git commit`, run `bash spikes/editor/scripts/check-root-clean.sh` (staged **and** unstaged check of the three root files). Task 7's final gate diffs the whole branch against the base SHA. A post-commit `git diff HEAD` is NOT sufficient — it cannot see a committed root mutation. (§10, fixes review F4)
- **No `any`:** never `any`, `@ts-ignore`, `@ts-expect-error`. Narrow `unknown`. A dedicated
  gate scans all spike-authored TS/TSX (client, tests, and scripts); browser-only boundary
  patterns remain scoped to candidate `src/`. Per-package `tsc --strict` is additional, not a
  substitute for rejecting explicit `any`. (§13)
- **Guarded file I/O:** every script that reads files uses `spikes/editor/scripts/lib/safe-io.mjs` — `lstat` before read (reject symlinks/non-regular), per-file + total byte caps, JSON behind an `unknown` guard, log path+reason only, never contents. (§6a, fixes review F3)
- **Boundary:** `@spike/oracle` (imports `@movp/domain`, `@supabase/*`) is Node-only; client source imports only `@spike/fixture`. (§6)
- **Exact-pin deps** with `pnpm add -E …`; every peer compatible with `react@18.3.1`. Pin `@supabase/supabase-js@2.110.0` — the version the ROOT lock resolves (verified) — so the oracle's `SupabaseClient` type matches the one `makeContentService` consumes via linked `@movp/domain`. "Same major" is not enough. (§11, fixes review F8)
- **`@movp/domain` is raw TS source** (`exports: "./src/index.ts"`, internal `.ts` specifiers). Any tsconfig type-checking code that imports it needs `"moduleResolution": "bundler"`, `"allowImportingTsExtensions": true`, `"noEmit": true`. Code that *runs* it goes through Vitest/esbuild, never `node` on emitted JS. (verified: `packages/domain/package.json`)
- **`richtext` is stored as an opaque `z.string()`**; production `canonicalize` sorts only outer `data` keys, so inner byte-stability is the adapter's job. (§2, §5.2; `packages/domain/src/content.ts:63–72`)
- **Every candidate gate feeds one machine-validated `CandidateResult`** (§12.2);
  selection requires every technical pass field plus the SPDX gate, while
  `blockIdPreserved` and declared-only notice gaps remain recorded trade-offs. A result
  file is written only after its owning assertions pass, and the clean-run deletes every
  `.report/` first so stale evidence cannot false-pass. (fixes review F2/F7)
- **TDD:** every task is failing-test → run-fail → implement → run-pass → commit.
  After every individual file-edit patch, rerun the narrowest current task gate and show
  its output; an intermediate expected failure is acceptable, but an untested edit is not.

/** §5.2 normative canonical inner-JSON algorithm. Byte-stable string for a JSON value. */
export function canonicalizeInnerJson(value: unknown): string {
  return serialize(value, new WeakSet<object>())
}

function isPlainObject(v: object): v is Record<string, unknown> {
  const proto = Object.getPrototypeOf(v)
  return proto === Object.prototype || proto === null
}

function serialize(value: unknown, ancestors: WeakSet<object>): string {
  if (value === null) return 'null'
  const t = typeof value
  if (t === 'string') return JSON.stringify(value)
  if (t === 'boolean') return value ? 'true' : 'false'
  if (t === 'number') {
    if (!Number.isFinite(value)) throw new Error('canonical: non-finite number rejected')
    return JSON.stringify(value)
  }
  if (t === 'bigint') throw new Error('canonical: bigint rejected')
  if (t === 'undefined') throw new Error('canonical: undefined rejected')
  if (Array.isArray(value)) {
    if (ancestors.has(value)) throw new Error('canonical: cycle rejected')
    ancestors.add(value)
    try {
      const items: string[] = []
      for (let index = 0; index < value.length; index += 1) {
        items.push(serialize(value[index], ancestors))
      }
      return `[${items.join(',')}]`
    } finally {
      ancestors.delete(value)
    }
  }
  if (t === 'object') {
    const obj = value as object
    if (!isPlainObject(obj)) throw new Error('canonical: non-plain object rejected')
    if (ancestors.has(obj)) throw new Error('canonical: cycle rejected')
    ancestors.add(obj)
    try {
      const keys = Object.keys(obj).sort()
      return `{${keys.map((k) => `${JSON.stringify(k)}:${serialize(obj[k], ancestors)}`).join(',')}}`
    } finally {
      ancestors.delete(obj)
    }
  }
  throw new Error(`canonical: unsupported value of type ${t}`)
}
```

import { chmodSync, lstatSync, readdirSync, readFileSync, renameSync, unlinkSync, writeFileSync } from 'node:fs'
import { randomUUID } from 'node:crypto'
import { dirname, join } from 'node:path'

const MAX_FILE_BYTES = 5 * 1024 * 1024   // per-file cap
const MAX_TOTAL_BYTES = 64 * 1024 * 1024 // walk total cap

// lstat (does NOT follow symlinks); reject symlinks + non-regular files; bound size.
function assertRegularBounded(path, maxBytes = MAX_FILE_BYTES) {
  const st = lstatSync(path) // throws if missing
  if (st.isSymbolicLink()) throw new Error(`safe-io: symlink rejected: ${path}`)
  if (!st.isFile()) throw new Error(`safe-io: non-regular file rejected: ${path}`)
  if (st.size > maxBytes) throw new Error(`safe-io: file exceeds cap: ${path}`)
  return st
}

export function assertSafeDirectory(path) {
  const st = lstatSync(path)
  if (st.isSymbolicLink()) throw new Error(`safe-io: symlink rejected: ${path}`)
  if (!st.isDirectory()) throw new Error(`safe-io: non-directory rejected: ${path}`)
}

export function readTextBounded(path, maxBytes = MAX_FILE_BYTES) {
  assertRegularBounded(path, maxBytes)
  return readFileSync(path, 'utf8')
}

export function readJsonBounded(path, maxBytes = MAX_FILE_BYTES) {
  const text = readTextBounded(path, maxBytes)
  let parsed
  try { parsed = JSON.parse(text) } catch { throw new Error(`safe-io: invalid JSON: ${path}`) }
  return parsed // returns unknown; caller must type-guard
}

export function writeTextAtomic(path, text, mode = 0o600) {
  if (typeof text !== 'string') throw new Error(`safe-io: write requires text: ${path}`)
  assertSafeDirectory(dirname(path))
  try {
    const st = lstatSync(path)
    if (st.isSymbolicLink() || !st.isFile()) throw new Error(`safe-io: unsafe write target: ${path}`)
  } catch (error) {
    if (!(error instanceof Error) || !('code' in error) || error.code !== 'ENOENT') throw error
  }
  const temp = `${path}.${randomUUID()}.tmp`
  try {
    writeFileSync(temp, text, { encoding: 'utf8', mode: 0o600, flag: 'wx' })
    chmodSync(temp, mode)
    renameSync(temp, path)
  } catch (error) {
    try { unlinkSync(temp) } catch { /* best-effort cleanup; original error remains authoritative */ }
    throw error
  }
}

export function writeJsonAtomic(path, value) {
  writeTextAtomic(path, `${JSON.stringify(value, null, 2)}\n`)
}

/** Yields regular-file paths under dir; rejects symlinked entries; skips node_modules/dist. */
export function walkRegularFiles(dir) {
  const out = []
  let total = 0
  const visit = (d) => {
    assertSafeDirectory(d) // guard this directory before every enumeration, including the root
    for (const name of readdirSync(d)) {
      if (name === 'node_modules' || name === 'dist') continue
      const p = join(d, name)
      const st = lstatSync(p)
      if (st.isSymbolicLink()) throw new Error(`safe-io: symlink rejected: ${p}`)
      if (st.isDirectory()) { visit(p); continue }
      if (!st.isFile()) throw new Error(`safe-io: non-regular file rejected: ${p}`)
      total += st.size
      if (total > MAX_TOTAL_BYTES) throw new Error('safe-io: total byte cap exceeded')
      out.push(p)
    }
  }
  visit(dir)
  return out
}

#!/usr/bin/env node
import { execFileSync } from 'node:child_process'
import { createHash } from 'node:crypto'
import { existsSync } from 'node:fs'
import { join, resolve } from 'node:path'
import { assertSafeDirectory, readJsonBounded, readTextBounded, writeTextAtomic } from './lib/safe-io.mjs'

const dir = join(process.cwd(), '.report')
try { assertSafeDirectory(dir) } catch (error) {
  console.error(error instanceof Error ? error.message : 'report: unknown directory failure')
  process.exit(1)
}
const load = (f) => { const p = join(dir, f); if (!existsSync(p)) { console.error(`report: ${p} missing`); process.exit(1) } return readJsonBounded(p) }
const isObj = (o) => typeof o === 'object' && o !== null

const TECHNICAL_PASS = ['idempotent', 'exactEdit', 'lifecycleOrder', 'publishedRead', 'staleSabotage', 'boundary', 'a11y']
const STRUCTURAL_BOOLS = [...TECHNICAL_PASS, 'blockIdPreserved']
function validLicenses(value) {
  return Array.isArray(value) && value.length > 0 && value.every((entry) =>
    isObj(entry) && typeof entry.name === 'string' && typeof entry.license === 'string' &&
    Array.isArray(entry.versions) && entry.versions.length > 0 && entry.versions.every((version) => typeof version === 'string'))
}
function validNotices(value) {
  return Array.isArray(value) && value.length > 0 && value.every((entry) =>
    isObj(entry) && typeof entry.package === 'string' &&
    ((entry.status === 'file' && typeof entry.path === 'string' && typeof entry.sha256 === 'string' && /^[0-9a-f]{64}$/.test(entry.sha256)) ||
     (entry.status === 'declared_only' && typeof entry.declaredLicense === 'string')))
}
function validateCandidate(value, expected) {
  if (!isObj(value) || value.schemaVersion !== 1 || value.candidate !== expected ||
      !isObj(value.resolvedVersions) || Object.keys(value.resolvedVersions).length === 0 ||
      !Object.values(value.resolvedVersions).every((version) => typeof version === 'string' && /^\d+\.\d+\.\d+(?:-[0-9A-Za-z.-]+)?$/.test(version)) ||
      !STRUCTURAL_BOOLS.every((key) => typeof value[key] === 'boolean') || !['pass', 'fail'].includes(value.license) ||
      typeof value.prodHasCopyleft !== 'boolean' || !validLicenses(value.prodLicenses) ||
      !validLicenses(value.fullLicenses) || !validNotices(value.noticeEvidence) ||
      !isObj(value.bundle) || !['jsRaw', 'jsGzip', 'cssRaw', 'cssGzip'].every((key) =>
        typeof value.bundle[key] === 'number' && Number.isFinite(value.bundle[key]) && value.bundle[key] >= 0) ||
      value.bundle.jsRaw === 0 || value.bundle.jsGzip === 0 ||
      (expected === 'tiptap' && (typeof value.toolbarLoc !== 'number' || !Number.isInteger(value.toolbarLoc) || value.toolbarLoc <= 0))) {
    console.error(`report: malformed ${expected} CandidateResult`); process.exit(1)
  }
  const measuredCopyleft = value.prodLicenses.some((entry) => ['MPL-2.0', 'EPL-2.0', 'GPL-2.0-only', 'GPL-3.0-only', 'AGPL-3.0-only', 'LGPL-2.1-only', 'LGPL-3.0-only'].includes(entry.license))
  if (measuredCopyleft !== value.prodHasCopyleft) {
    console.error(`report: ${expected} prodHasCopyleft disagrees with measured licenses`); process.exit(1)
  }
  return value
}

const bn = validateCandidate(load('blocknote.json'), 'blocknote')
const tt = validateCandidate(load('tiptap.json'), 'tiptap')
const decPath = join(dir, 'decision.json')
if (!existsSync(decPath)) { console.error('report: .report/decision.json missing — record licenseDecision + selectedCandidate'); process.exit(1) }
const decision = readJsonBounded(decPath)

const KINDS = ['accept_mpl', 'permissive_only'], CANDS = ['blocknote', 'tiptap']
if (!isObj(decision) || !isObj(decision.licenseDecision) || !KINDS.includes(decision.licenseDecision.kind)) { console.error('report: licenseDecision.kind invalid'); process.exit(1) }
if (!CANDS.includes(decision.selectedCandidate)) { console.error('report: selectedCandidate invalid'); process.exit(1) }

const chosen = decision.selectedCandidate === 'blocknote' ? bn : tt
for (const k of TECHNICAL_PASS) if (chosen?.[k] !== true) { console.error(`report: selected ${decision.selectedCandidate} field ${k} not passing`); process.exit(1) }
if (chosen?.license !== 'pass') { console.error(`report: selected ${decision.selectedCandidate} license not pass`); process.exit(1) }

if (decision.licenseDecision.kind === 'permissive_only' && chosen.prodHasCopyleft) { console.error(`report: inconsistent — permissive_only but ${decision.selectedCandidate} prod graph has copyleft`); process.exit(1) }

console.log(`report: ok — selected ${decision.selectedCandidate} under ${decision.licenseDecision.kind}`)
if (!process.argv.includes('--validate-only')) {
  const spikeRoot = process.cwd()
  const repoRoot = resolve(spikeRoot, '../..')
  const base = readTextBounded(join(spikeRoot, '.spike-base-sha'), 256).trim()
  if (!/^[0-9a-f]{40}$/.test(base)) { console.error('report: invalid base SHA'); process.exit(1) }
  const head = execFileSync('git', ['rev-parse', 'HEAD'], { cwd: repoRoot, encoding: 'utf8' }).trim()
  const lockDigest = createHash('sha256').update(readTextBounded(join(spikeRoot, 'pnpm-lock.yaml'), 16 * 1024 * 1024)).digest('hex')
  const json = (value) => JSON.stringify(value, null, 2)
  const markdown = [
    '<!-- Generated by pnpm --dir spikes/editor run report. Do not edit by hand. -->',
    '# C7.1 Editor Dependency Spike Report', '',
    `- Selected candidate: **${decision.selectedCandidate}**`,
    `- License decision: **${decision.licenseDecision.kind}**`,
    `- Execution base: \`${base}\``, `- Execution head: \`${head}\``,
    `- Nested lockfile SHA-256: \`${lockDigest}\``, '',
    '| Candidate | Idempotent | Exact edit | Delivery | Boundary | A11y | Block IDs preserved (informational) | Prod copyleft | JS raw/gzip | CSS raw/gzip |',
    '| --- | --- | --- | --- | --- | --- | --- | --- | --- | --- |',
    `| BlockNote | ${bn.idempotent} | ${bn.exactEdit} | ${bn.publishedRead} | ${bn.boundary} | ${bn.a11y} | ${bn.blockIdPreserved} | ${bn.prodHasCopyleft} | ${bn.bundle.jsRaw}/${bn.bundle.jsGzip} | ${bn.bundle.cssRaw}/${bn.bundle.cssGzip} |`,
    `| TipTap | ${tt.idempotent} | ${tt.exactEdit} | ${tt.publishedRead} | ${tt.boundary} | ${tt.a11y} | ${tt.blockIdPreserved} | ${tt.prodHasCopyleft} | ${tt.bundle.jsRaw}/${tt.bundle.jsGzip} | ${tt.bundle.cssRaw}/${tt.bundle.cssGzip} |`, '',
    '## BlockNote evidence', '', '```json', json(bn), '```', '',
    '## TipTap evidence', '', '```json', json(tt), '```', '',
    '## Terminal decision', '', '```json', json(decision), '```', '',
  ].join('\n')
  const output = join(repoRoot, 'docs/superpowers/specs/2026-07-15-c7.1-editor-spike-report.md')
  writeTextAtomic(output, markdown, 0o644)
  console.log(`report: wrote ${output}`)
}
```
