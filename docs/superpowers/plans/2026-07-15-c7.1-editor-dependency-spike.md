# C7.1 Editor Dependency Spike — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Decide BlockNote vs TipTap for Phase C7 by proving, head-to-head, that a candidate can round-trip MOVP `content_revision.data` hash-stably through the real create → **editor-driven edit** → publish → published-read lifecycle, within the client/server boundary, at an acceptable license.

**Architecture:** A throwaway nested pnpm workspace `spikes/editor/` (outside the root workspace globs) with four packages: `@spike/fixture` (browser-safe data + the normative canonical algorithm), `@spike/oracle` (Node-only; runs the real `@movp/domain` hash path through a real `SupabaseClient` wired to a stateful fake `fetch`), and `@spike/blocknote` / `@spike/tiptap` (equivalent React editor islands + Playwright-mounts/Node-hashes proofs, boundary gates, a11y, and a machine-validated `CandidateResult`).

**Tech Stack:** pnpm 9.12.0 nested workspace; TypeScript (bundler resolution); Vitest (Node); Playwright + `@axe-core/playwright`; Vite (islands + `getModuleIds` boundary artifact); BlockNote (`@blocknote/*` + Mantine) vs TipTap (`@tiptap/*` + StarterKit).

**Source spec:** `docs/superpowers/specs/2026-07-15-c7.1-editor-spike-design.md`
(land this revised spec and plan together). Section refs below (§N) point at it.

## Global Constraints

- **Isolation, checked BEFORE every commit and against a base SHA.** `spikes/` is NOT in root `pnpm-workspace.yaml`. Task 1 records the pre-spike commit in `spikes/editor/.spike-base-sha`. Before every `git commit`, run `bash spikes/editor/scripts/check-root-clean.sh` (staged **and** unstaged check of the three root files). Task 7's final gate diffs the whole branch against the base SHA. A post-commit `git diff HEAD` is NOT sufficient — it cannot see a committed root mutation. (§10, fixes review F4)
- **No `any`:** never `any`, `@ts-ignore`, `@ts-expect-error`. Narrow `unknown`. A dedicated
  gate scans all spike-authored TS/TSX (client, tests, and scripts); browser-only boundary
  patterns remain scoped to candidate `src/`. Per-package `tsc --strict` is additional, not a
  substitute for rejecting explicit `any`. (§13)
- **Guarded file I/O:** every script that reads files uses `spikes/editor/scripts/lib/safe-io.mjs` — `lstat` before read (reject symlinks/non-regular), per-file + total byte caps, JSON behind an `unknown` guard, log path+reason only, never contents. (§6a, fixes review F3)
- **Boundary:** `@spike/oracle` (imports `@movp/domain`, `@supabase/*`) is Node-only; client source imports only `@spike/fixture`. (§6)
- **Exact-pin deps** with `pnpm add -E …`; every peer compatible with `react@18.3.1`. Pin `@supabase/supabase-js@2.110.0` — the version the ROOT lock resolves (verified) — so the oracle's `SupabaseClient` type matches the one `makeContentService` consumes via linked `@movp/domain`. "Same major" is not enough. (§11, fixes review F8)
- **Pinned runtime:** `spikes/editor/.node-version` is exactly `22.23.0`; candidate engines
  are exactly `22.23.0`. Candidate `test` and `report` invoke the shared fail-loud runtime preflight,
  which launches Playwright's signed `channel: 'chrome'` without a hardcoded host path and
  records Node/browser versions. Select Node 22.23.0 before any clean run. On macOS, do not
  strip xattrs or weaken provenance checks to make bundled unsigned Chromium launch. (§10)
- **`@movp/domain` is raw TS source** (`exports: "./src/index.ts"`, internal `.ts` specifiers). Any tsconfig type-checking code that imports it needs `"moduleResolution": "bundler"`, `"allowImportingTsExtensions": true`, `"noEmit": true`. Code that *runs* it goes through Vitest/esbuild, never `node` on emitted JS. (verified: `packages/domain/package.json`)
- **`richtext` is stored as an opaque `z.string()`**; production `canonicalize` sorts only outer `data` keys, so inner byte-stability is the adapter's job. (§2, §5.2; `packages/domain/src/content.ts:63–72`)
- **Every candidate gate feeds one machine-validated `CandidateResult`** (§12.2).
  Candidate evaluators pass by asserting the exact measured classification, including an
  attributable `false`; evaluator PASS is not candidate PASS. A technical boolean is written only after
  its owning assertion pins that measured outcome. Final selection still requires every
  technical pass field plus the SPDX gate, while `blockIdPreserved` and declared-only notice
  gaps remain recorded trade-offs. The clean-run deletes every `.report/` first so stale
  evidence cannot false-pass. (fixes review F2/F7; negative-candidate protocol)
- **TDD:** every task is failing-test → run-fail → implement → run-pass → commit.
  After every individual file-edit patch, rerun the narrowest current task gate and show
  its output; an intermediate expected failure is acceptable, but an untested edit is not.

```ts
/** §5.2 normative canonical inner-JSON algorithm. Byte-stable string for a JSON value. */
export function canonicalizeInnerJson(value: unknown): string {
  return serialize(value, new WeakSet<object>())
}

function isPlainObject(v: object): v is Record<string, unknown> {
  const proto = Object.getPrototypeOf(v)
  return proto === Object.prototype || proto === null
}

function serialize(value: unknown, ancestors: WeakSet<object>): string {
  if (value === null) return 'null'
  const t = typeof value
  if (t === 'string') return JSON.stringify(value)
  if (t === 'boolean') return value ? 'true' : 'false'
  if (t === 'number') {
    if (!Number.isFinite(value)) throw new Error('canonical: non-finite number rejected')
    return JSON.stringify(value)
  }
  if (t === 'bigint') throw new Error('canonical: bigint rejected')
  if (t === 'undefined') throw new Error('canonical: undefined rejected')
  if (Array.isArray(value)) {
    if (ancestors.has(value)) throw new Error('canonical: cycle rejected')
    ancestors.add(value)
    try {
      const items: string[] = []
      for (let index = 0; index < value.length; index += 1) {
        items.push(serialize(value[index], ancestors))
      }
      return `[${items.join(',')}]`
    } finally {
      ancestors.delete(value)
    }
  }
  if (t === 'object') {
    const obj = value as object
    if (!isPlainObject(obj)) throw new Error('canonical: non-plain object rejected')
    if (ancestors.has(obj)) throw new Error('canonical: cycle rejected')
    ancestors.add(obj)
    try {
      const keys = Object.keys(obj).sort()
      return `{${keys.map((k) => `${JSON.stringify(k)}:${serialize(obj[k], ancestors)}`).join(',')}}`
    } finally {
      ancestors.delete(obj)
    }
  }
  throw new Error(`canonical: unsupported value of type ${t}`)
}
```

```js
import { chmodSync, lstatSync, readdirSync, readFileSync, renameSync, unlinkSync, writeFileSync } from 'node:fs'
import { randomUUID } from 'node:crypto'
import { dirname, join } from 'node:path'

const MAX_FILE_BYTES = 5 * 1024 * 1024   // per-file cap
const MAX_TOTAL_BYTES = 64 * 1024 * 1024 // walk total cap

// lstat (does NOT follow symlinks); reject symlinks + non-regular files; bound size.
function assertRegularBounded(path, maxBytes = MAX_FILE_BYTES) {
  const st = lstatSync(path) // throws if missing
  if (st.isSymbolicLink()) throw new Error(`safe-io:E_SYMLINK path=${path}`)
  if (!st.isFile()) throw new Error(`safe-io:E_NOT_REGULAR path=${path}`)
  if (st.size > maxBytes) throw new Error(`safe-io:E_FILE_CAP path=${path}`)
  return st
}

export function assertSafeDirectory(path) {
  const st = lstatSync(path)
  if (st.isSymbolicLink()) throw new Error(`safe-io:E_SYMLINK path=${path}`)
  if (!st.isDirectory()) throw new Error(`safe-io:E_NOT_DIRECTORY path=${path}`)
}

export function readTextBounded(path, maxBytes = MAX_FILE_BYTES) {
  assertRegularBounded(path, maxBytes)
  return readFileSync(path, 'utf8')
}

export function readJsonBounded(path, maxBytes = MAX_FILE_BYTES) {
  const text = readTextBounded(path, maxBytes)
  let parsed
  try { parsed = JSON.parse(text) } catch { throw new Error(`safe-io:E_INVALID_JSON path=${path}`) }
  return parsed // returns unknown; caller must type-guard
}

export function writeTextAtomic(path, text, mode = 0o600) {
  if (typeof text !== 'string') throw new Error(`safe-io:E_TEXT_REQUIRED path=${path}`)
  assertSafeDirectory(dirname(path))
  try {
    const st = lstatSync(path)
    if (st.isSymbolicLink() || !st.isFile()) throw new Error(`safe-io:E_UNSAFE_TARGET path=${path}`)
  } catch (error) {
    if (!(error instanceof Error) || !('code' in error) || error.code !== 'ENOENT') throw error
  }
  const temp = `${path}.${randomUUID()}.tmp`
  try {
    writeFileSync(temp, text, { encoding: 'utf8', mode: 0o600, flag: 'wx' })
    renameSync(temp, path)
    chmodSync(path, mode)
  } catch (error) {
    try { unlinkSync(temp) } catch { /* best-effort cleanup; original error remains authoritative */ }
    throw error
  }
}

export function writeJsonAtomic(path, value) {
  writeTextAtomic(path, `${JSON.stringify(value, null, 2)}\n`)
}

/** Yields regular-file paths under dir; rejects symlinked entries; skips node_modules/dist. */
export function walkRegularFiles(dir) {
  const out = []
  let total = 0
  const visit = (d) => {
    assertSafeDirectory(d) // guard this directory before every enumeration, including the root
    for (const name of readdirSync(d)) {
      if (name === 'node_modules' || name === 'dist') continue
      const p = join(d, name)
      const st = lstatSync(p)
      if (st.isSymbolicLink()) throw new Error(`safe-io:E_SYMLINK path=${p}`)
      if (st.isDirectory()) { visit(p); continue }
      if (!st.isFile()) throw new Error(`safe-io:E_NOT_REGULAR path=${p}`)
      total += st.size
      if (total > MAX_TOTAL_BYTES) throw new Error(`safe-io:E_TOTAL_CAP path=${dir} count=${total}`)
      out.push(p)
    }
  }
  visit(dir)
  return out
}
```
